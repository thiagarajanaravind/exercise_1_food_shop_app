# exercise_1_food_shop_app
Kotlin Jetpack Compose Food Shop App created in Android Studio.

Exercise 1 - Kotlin Jetpack Compose Food Shop App

1. Tools Used
● Android Studio Panda 2025.3.4 Patch 1
● Kotlin
● Jetpack Compose
● Pixel 7 Emulator
● Minimum SDK: API 24

2. How to Run the Project
● Extract the Exercise_1 zip file.
● Open Android Studio.
● Click Open and select the Exercise_1 project folder.
● Wait for Gradle Sync to finish.
● Select Pixel 7 emulator or another Android emulator.
● Click the Run button.
● The app will start on the emulator.

3. App Features
● Home screen with food image and Start Shopping button.
● Shop screen with scrollable food cards.
● Add to Cart button for each food item.
● Cart screen showing selected items.
● Total price calculation.
● Purchase button.
